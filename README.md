# Dashboard IPA for TrollStore

Clock + Calendar + To-Do List dashboard for iPad Air 2 (iOS 15).

## How to build

1. Fork or clone this repo to your GitHub account
2. Go to **Actions** tab → **Build Dashboard IPA** → **Run workflow**
3. Wait ~5 minutes for build to finish
4. Download `Dashboard-IPA` from the Artifacts section
5. Extract the zip → get `Dashboard.ipa`
6. Install via **TrollStore** on your iPad
